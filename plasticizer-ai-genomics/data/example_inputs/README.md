# Example Dataset

This folder contains **small representative example files** provided solely for testing, demonstration, and reproducibility of the computational workflows associated with the plasticizer biodegradation genomics and AI enzyme prediction project.

These example datasets allow users to run the analysis pipelines, generate figures, and verify software installation without requiring access to large sequencing datasets or full training data.

---

## Purpose of Example Data

The files in this directory are intended to:

* Demonstrate feature extraction workflows
* Test machine-learning pipelines
* Validate genome visualization scripts
* Reproduce example plots and outputs
* Confirm correct environment and dependency setup

These datasets are intentionally reduced in size and **should not be used for scientific analysis or biological interpretation**.

---

## Typical Contents

Depending on the workflow, example files may include:

* Example protein FASTA files for testing AI prediction pipelines
* Sample Pfam domain annotation outputs (`.domtblout`)
* Example candidate enzyme scoring tables
* ROC evaluation example datasets
* Feature importance example outputs
* Small genome annotation subsets for visualization testing

## Reproducibility

To reproduce analyses:

1. Install dependencies using `environment.yml` or `requirements.txt`
2. Run scripts in the `src/` directory
3. Use example data here for testing or download full datasets from Figshare.
